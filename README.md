# Auth Service

This is a Spring Boot application written in Kotlin that handles **user authentication** with JWT tokens.

## Features

- Sign-up and Sign-in
- Password hashing
- JWT-based authentication
- PostgreSQL database
- Flyway for database migrations
- jOOQ for type-safe SQL queries
